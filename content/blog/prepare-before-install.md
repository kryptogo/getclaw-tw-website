每次到府安裝前，客戶最常問的一句話就是：**「需要我先準備什麼？」**

答案是：如果你什麼都沒準備，我們當天還是能裝——但會花不少時間在等你申請帳號、找密碼、確認權限。這些等待時間加起來，輕鬆吃掉一個小時。

這篇文章把所有該準備的東西列出來。**花 15 分鐘提前搞定，當天就能直接進入重點：設計你的 AI 工作流。**

---

## 一、硬體 & 網路

AI 助理需要一台「常開」的電腦來執行。你不需要買新設備，但需要確保有一台可以 24 小時運作的機器。

### 必備

- **一台可以常開的電腦**
  - Mac mini 是最推薦的選擇（安靜、省電、效能夠用）
  - 舊的 MacBook / Windows 筆電也行，只要能穩定開機
  - NAS（如 Synology）可以跑 Docker 的機型也可以
  - 如果你完全沒有閒置設備，我們可以幫你代購 Mac mini（約 NT$19,900 起）
- **穩定的網路連線**
  - 需要能連到外部 API（Anthropic / OpenAI）
  - 不需要固定 IP，一般家用或辦公室網路就夠
  - 如果公司有防火牆或 VPN 限制，請提前告知 IT 部門開放相關連線

### 加分但非必要

- 如果你希望從外部存取（例如在家也能用），可以準備：
  - Tailscale 帳號（免費方案就夠）
  - 或者公司有 VPN 存取方式

## 二、帳號申請（最容易被卡住的部分）

**這是最常拖延安裝進度的環節。** API Key 的申請流程有時需要驗證信用卡、等待審核，所以請務必提前完成。

### AI 模型 API Key（擇一即可）

你的 AI 助理需要透過 API 呼叫大型語言模型。以下擇一申請：

| 供應商 | 申請連結 | 建議方案 | 備註 |
|--------|---------|---------|------|
| **Anthropic（Claude）** | [console.anthropic.com](https://console.anthropic.com) | 儲值 US$20 起 | 推薦，中文能力優秀 |
| **OpenAI（GPT）** | [platform.openai.com](https://platform.openai.com) | 儲值 US$20 起 | 生態系最完整 |

**申請步驟（兩家流程類似）：**
1. 註冊帳號
2. 進入 API Keys 頁面，建立一組新的 API Key
3. **把 Key 複製下來存好**（只會顯示一次）
4. 綁定信用卡並儲值（建議先儲 US$20，夠用一個月以上）

如果兩家都不確定選哪個，**我們建議先申請 Anthropic**。到現場後我們會根據你的使用場景幫你決定。

如果你選擇使用 ClawCare 方案，Token 費用全包在月費裡，不需要自己管 API Key。但導入階段還是需要先有一組 Key 來測試。

### 通訊軟體 Bot（你的 AI 助理「住」在哪裡）

AI 助理需要一個跟你對話的介面。**選你最常用的那個就好：**

| 平台 | 適合誰 | 準備方式 |
|------|--------|---------|
| **LINE** | 台灣多數人的首選 | 我們會幫你設定 LINE Official Account（需要你的 LINE 帳號登入） |
| **Telegram** | 重度使用者 / 工程師 | 用 @BotFather 建立一個 Bot，拿到 Bot Token |
| **Discord** | 團隊協作 / 社群 | 在你的 Server 建立一個 Bot Application |
| **Slack** | 企業團隊 | 需要 Workspace Admin 權限來安裝 App |

**不確定選哪個？選 LINE。** 你每天都在用，學習成本最低。

## 三、Google 帳號權限（如需整合）

大多數客戶會想整合 Gmail 和 Google Calendar。如果你也需要，請提前確認：

- [ ] 你的 Google 帳號可以登入（記得密碼、通過兩步驟驗證）
- [ ] 如果是公司的 Google Workspace 帳號，確認管理員允許第三方應用存取
- [ ] 如果你想整合多個信箱，每個都需要授權

**常見卡點：** 公司 IT 把第三方 OAuth 鎖住了。如果你不確定，請提前問一下公司 IT：「我能不能授權一個自架的應用程式存取我的 Gmail？」

## 四、想清楚你的需求（最重要的準備）

硬體和帳號都是技術準備，但**真正影響導入成效的，是你有沒有想過 AI 可以幫你做什麼。**

### 請花 10 分鐘想以下問題：

**1. 你每天最煩的 3 件重複性工作是什麼？**

不需要想得很具體，例如：
- 「每天花一小時處理 email」
- 「每週要做一份週報，每次都很痛苦」
- 「客戶問的問題都差不多，但我每次都要重新回」

**2. 你有沒有一些「知道應該做但一直沒時間做」的事？**

例如：
- 「應該要定期整理客戶資料，但一直拖著」
- 「想經營社群但沒時間寫文」
- 「會議紀錄都沒在做」

**3. 你的資料隱私考量**

這會直接影響我們的部署方式：
- 資料可以上雲端嗎？還是必須留在本地？
- 有沒有客戶資料、財務資料等敏感內容會經過 AI？
- 公司有沒有資安政策需要遵守？

### 如果你能準備這些，會更好：

- **一週份的工作信件**（讓我們了解你的溝通模式）
- **你常用的文件範本**（報價單、合約、報告等）
- **團隊的組織架構**（如果要設定多個助理角色）

## 五、工程師加分題

如果你本身是工程師或有技術背景，以下可以先準備：

- [ ] GitHub 帳號
- [ ] 已安裝 Docker Desktop（或 OrbStack）
- [ ] 已安裝 Node.js 18+
- [ ] 已安裝 Git
- [ ] SSH Key 已設定好

這些都不是必要的——我們到場會幫你處理。但如果你已經有，能省下不少安裝時間。

## 完整檢查清單

安裝前一天，對照這個清單確認一次：

- [ ] 有一台可以常開的電腦，已開機並連上網路
- [ ] AI API Key 已申請好，Key 已複製存好
- [ ] 已儲值至少 US$20
- [ ] 通訊軟體 Bot 已選定（LINE / Telegram / Discord / Slack）
- [ ] Google 帳號可以登入（如需整合）
- [ ] 想好了最想讓 AI 幫你做的 3 件事
- [ ] 確認過資料隱私需求

## 當天流程預覽

準備好以上這些之後，到府當天的流程大概是這樣：

1. **前 15 分鐘**：聊你的工作方式和需求，確認 AI 助理的角色配置
2. **接下來 1-2 小時**：安裝系統、串接服務、建立你的前三個工作流程
3. **最後 15 分鐘**：一起測試、確認一切正常運作，交付文件

如果你提前準備好了，我們就能把時間花在最有價值的地方——**設計真正適合你的 AI 工作流**，而不是在等你找密碼。
