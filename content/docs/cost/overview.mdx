很多人以為自架 AI Agent 最花錢的是伺服器。錯了。**API 呼叫費用才是真正的成本黑洞**。搞不清楚成本結構就開始架，三個月後看到帳單才後悔，這種故事我們聽太多了。

## 三大成本類別

自架 AI Agent 的成本可以拆成三塊：

### 1. 基礎設施（VPS / 硬體）

這是最可預測的一塊。一台基本的 VPS 每月大約 NT$300，一年 NT$3,600。如果你用 Mac Mini 放在家裡跑，硬體攤提後每月成本更低。

這不是重點。往下看。

### 2. AI API 呼叫費用

這才是重頭戲。不同模型的價格差距極大：

| 模型 | 輸入成本（每百萬 token） | 輸出成本（每百萬 token） | 等級 |
|------|--------------------------|--------------------------|------|
| MiniMax / K2.5 | ~$0.15 | ~$0.55 | 日常 |
| Claude Sonnet | ~$3 | ~$15 | 進階 |
| GPT-4o | ~$2.50 | ~$10 | 進階 |
| Claude Opus | ~$15 | ~$75 | 關鍵 |

一個日常對話大約消耗 2,000-5,000 tokens。如果你全程用 Opus 級模型，每天聊 20 回合，一個月的 API 帳單可能超過 NT$1,500。但如果你用對策略，同樣的使用量可以壓到 NT$100 以內。

**差距是 15 倍。**

### 3. 維護人力

自架不是設定完就沒事了。系統更新、模型切換、問題排除都需要時間。但 OpenClaw 的設計目標就是把這塊降到最低 — 大部分維護工作可以透過 TUI 或指令完成，不需要你是 Linux 專家。

## 12 個月 TCO 比較

把三種方案攤開來算：

| 方案 | 年度成本 | 使用限制 | 模型選擇 |
|------|----------|----------|----------|
| SaaS（如 ChatGPT Plus） | ~NT$7,800/年 | 有用量上限、降速 | 僅該平台模型 |
| 自架基本版 | VPS ~NT$3,600/年 + API 變動 | 無上限 | 任意模型 |
| 自架最佳化版 | VPS + OAuth 策略 | 依訂閱方案 | 多模型切換 |

SaaS 方案看起來簡單，但你付的是「方便稅」。每月 NT$650 買到的是有上限的使用量和單一模型。一旦你的使用量超過免費額度，就會被降速或截斷。

自架基本版的固定成本更低，但 API 費用是變動的。如果不做任何最佳化，高頻使用者的 API 費用可能比 SaaS 還貴。

**關鍵在第三個選項：自架 + 成本最佳化。** 透過 OAuth 訂閱策略和平價模型路由，你可以把總成本壓到比 SaaS 更低，同時享有更多彈性。

## 真正的成本驅動因子

讓我們直說：**基礎設施不是問題，API 使用方式才是。**

同樣跑 OpenClaw，成本可以差 10 倍以上。差別在於：

- 你是否選對了模型（日常任務用平價模型）
- 你是否善用了 OAuth 訂閱額度
- 你是否設定了自動模型切換策略

接下來兩篇文章會分別介紹這兩個省錢策略：

1. **[OAuth 訂閱策略](/docs/cost/oauth-strategy)** — 把你已經在付的訂閱費用轉化為 OpenClaw 的算力
2. **[平價模型策略](/docs/cost/budget-models)** — 用模型分層 + OpenRouter 路由，把 API 成本砍 70% 以上

## GetClaw 觀點

自架 AI Agent 的經濟學不複雜，但很多人搞反了優先順序。他們花大量時間比較 VPS 方案，卻對 API 費用毫無策略。
