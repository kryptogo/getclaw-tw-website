## 什麼是 Antigravity？

Antigravity 是 Google 內部的 IDE 環境，透過 OAuth 認證可以使用：
- Claude Opus 4.5 / Sonnet 4.5（含 Thinking 模式）
- Gemini 3 Pro / Flash
- 更高的 Rate Limits

## 啟用 Antigravity Plugin

```bash
openclaw plugins enable google-antigravity-auth
```

## 登入 Antigravity

```bash
openclaw models auth login --provider google-antigravity --set-default
```

執行後會開啟瀏覽器，用你的 Google 帳號完成 OAuth 認證。

## 設定預設模型

編輯設定檔：

```bash
nano ~/.openclaw/openclaw.json
```

找到 `agents.defaults.model.primary`，改成你要的模型：

```json
"model": {
  "primary": "google-antigravity/claude-opus-4-5-thinking"
}
```

## 可用的 Antigravity 模型

### Claude 系列

| 模型 ID | 說明 |
|---------|------|
| `google-antigravity/claude-opus-4-5-thinking` | 最強推理能力 |
| `google-antigravity/claude-sonnet-4-5` | 平衡品質與速度 |
| `google-antigravity/claude-sonnet-4-5-thinking` | Sonnet + 深度思考 |

### Gemini 系列

| 模型 ID | 說明 |
|---------|------|
| `google-antigravity/gemini-3-pro-high` | 高品質 |
| `google-antigravity/gemini-3-pro-low` | 低延遲 |
| `google-antigravity/gemini-3-flash` | 快速回應 |

## 變更預設模型

```bash
nano ~/.openclaw/openclaw.json
```

找到 `"primary":` 那行，改成你想要的模型。儲存後重啟服務：

```bash
sudo systemctl restart openclaw
```

## 更新 Antigravity 版本

如果出現版本不支援的錯誤，需要手動更新版本號：

```bash
# 找到檔案位置
find /usr -name "google-gemini-cli.js" 2>/dev/null

# 編輯檔案（路徑可能不同）
sudo nano /usr/lib/node_modules/openclaw/node_modules/@mariozechner/pi-ai/dist/providers/google-gemini-cli.js
```

按 `Ctrl+W` 搜尋 `User-Agent`，把版本號從舊版改成最新版（例如 `1.11.5` → `1.15.8`）。儲存後重啟服務。

<Callout type="warn">
這是非官方的使用方式，可能違反 Google 服務條款。新帳號或剛訂閱的帳號有被封鎖的風險，建議使用非主要帳號。
</Callout>
