## 為什麼需要連接 GitHub？

讓你的 Bot 可以讀取和寫入私有倉庫，用於：
- 儲存 Bot 的記憶和設定
- 同步多個環境的資料
- 版本控制 Bot 的知識庫

## 步驟 1：在本機生成 SSH Key

```bash
ssh-keygen -t ed25519 -C "your-bot-name"
```

一路按 Enter 使用預設值。

## 步驟 2：將公鑰加到 GitHub

```bash
cat ~/.ssh/id_ed25519.pub
```

複製輸出的內容，然後：

1. 前往 GitHub → Settings → SSH and GPG keys
2. 點擊 **New SSH key**
3. Title 填入識別名稱（如 `VPS Bot`）
4. 貼上公鑰內容
5. 點擊 **Add SSH key**

## 步驟 3：在 VPS 上設定 SSH Key

```bash
# 建立 SSH 目錄
mkdir -p ~/.ssh
chmod 700 ~/.ssh

# 建立私鑰檔案
nano ~/.ssh/id_ed25519
# 貼上你的私鑰內容（從本機 cat ~/.ssh/id_ed25519 取得）

# 設定權限
chmod 600 ~/.ssh/id_ed25519
```

## 步驟 4：測試 SSH 連接

```bash
ssh -T git@github.com
```

第一次連接會詢問是否信任主機，輸入 `yes`。成功會顯示：

```
Hi <USERNAME>! You've successfully authenticated, but GitHub does not provide shell access.
```

## 步驟 5：克隆私有倉庫

```bash
git clone git@github.com:<USERNAME>/<REPO>.git
```
