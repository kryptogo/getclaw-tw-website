---
title: 安裝額外套件
description: 根據需求安裝 ffmpeg、yt-dlp、Playwright 等額外工具。
---

## ffmpeg

用於音訊和視訊處理：

```bash
sudo apt install -y ffmpeg
```

## yt-dlp

用於下載影片：

```bash
sudo apt install -y python3-pip
sudo pip3 install yt-dlp --break-system-packages
```

## Playwright + Chromium

用於瀏覽器自動化（截圖、網頁操作等）：

```bash
sudo npm install -g playwright
sudo playwright install chromium --with-deps
```

<Callout type="info">
Playwright + Chromium 會佔用較多記憶體。如果你的 VPS 只有 2GB RAM，建議在使用時才啟動，不要常駐運行。
</Callout>
