## 需要準備的東西

可根據進度逐步準備：

| 項目 | 說明 |
|------|------|
| VPS | Ubuntu 22.04 或 24.04，至少 2GB RAM |
| SSH 存取 | `ssh root@<VPS_IP>` |
| 網域 | 需要在 Cloudflare 管理（用於 Webhook） |
| Google 帳號 | 用於 Antigravity OAuth 認證 |
| LINE Channel | 從 [LINE Developers Console](https://developers.line.biz/) 取得 |

## 重要注意事項

<Callout type="warn">
以下三點是最常見的安裝失敗原因，請特別注意。
</Callout>

### 1. 套件名稱是 `openclaw`，不是 `moltbot`

因為改名了。正確安裝指令：

```bash
npm install -g openclaw@latest
```

### 2. Cloudflare 設定順序

- 必須**先**在 Cloudflare 新增網域
- **再**執行 `cloudflared tunnel login` 選擇正確的網域
- 順序錯誤會導致 DNS 記錄加到錯誤的網域

### 3. LINE 支援

- OpenClaw 有支援 LINE（需安裝 Plugin）
- 需要設定 Cloudflare Tunnel 提供 HTTPS Webhook URL
