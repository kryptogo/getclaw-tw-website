## 為什麼要建立新使用者？

- 比直接用 root 安全
- 平常用一般使用者，需要時才用 `sudo` 提升權限
- 業界標準做法

## 執行步驟

```bash
# 建立使用者（名稱可自訂）
adduser <USERNAME>
```

<Callout type="info">
Linux 輸入密碼不會顯示任何字元，這是正常的。密碼建議：至少 8 字元，混合英文 + 數字，例如 `MyPass2026!`。Full Name 等資訊可以直接按 Enter 跳過。
</Callout>

```bash
# 給予 sudo 權限
usermod -aG sudo <USERNAME>

# 確認權限
id <USERNAME>
# 應該看到 groups=...,27(sudo),... 表示成功
```

> 💡 **新使用者 + sudo 權限 = 需要時可以執行管理員指令**，ffmpeg、yt-dlp、Playwright 都能正常使用。
