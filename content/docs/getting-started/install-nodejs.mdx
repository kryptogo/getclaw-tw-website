---
title: 安裝 Node.js 22+
description: 安裝 OpenClaw 需要的 Node.js 22+ 執行環境。
---

## 切換到新使用者

```bash
su - <USERNAME>
```

## 安裝 Node.js

```bash
# 安裝必要套件
sudo apt update && sudo apt install -y ca-certificates curl gnupg

# 加入 NodeSource 來源
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -

# 安裝 Node.js
sudo apt install nodejs -y

# 確認版本
node -v && npm -v
# 應該顯示 v22.x.x 和 10.x.x
```

<Callout type="info">
如果 `node -v` 顯示的版本低於 22，請確認 NodeSource 來源已正確加入。可以重新執行 `curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -` 後再安裝。
</Callout>
