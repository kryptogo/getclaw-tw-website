---
title: SSH 連線到 VPS
description: 使用 SSH 連線到你的 VPS 主機。
---

## 連線指令

```bash
ssh root@<VPS_IP>
```

- 首次連線會詢問是否信任主機，輸入 `yes`
- 輸入 root 密碼

<Callout type="info">
如果你使用 macOS 或 Linux，終端機內建就有 SSH。Windows 使用者可以使用 PowerShell 或安裝 [Windows Terminal](https://aka.ms/terminal)。
</Callout>

## 確認連線成功

連線後你會看到類似這樣的提示：

```
root@vps-hostname:~#
```

這表示你已經以 root 身份登入 VPS，可以開始下一步。
